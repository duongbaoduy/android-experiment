To make reading of this page more valuable (for future search results) I made a list of libraries known to me.. As @CommonsWare mentioned there are super-similar questions/answers.. Anyway some libraries that can be used for making charts are:

Open Source:


* [Holo Graph Library](https://bitbucket.org/danielnadeau/holographlibrary/wiki/Home)
* [aChartEngine](http://www.achartengine.org/content/demo.html)
* [ChartView](https://github.com/nadavfima/ChartView/)
* [aFreeChart](http://code.google.com/p/afreechart/)
* [ChartDroid](http://code.google.com/p/chartdroid/)
* [charts4j](http://code.google.com/p/charts4j/)
* [GraphView](http://android.arnodenhond.com/components/graphview)
* [AndroidPlot](https://bitbucket.org/androidplot/androidplot/src)
* [Drawing the 3D piechart Using Google chart Api](http://androidtutorials60.blogspot.in/2013/06/drawing-3d-piechart-using-google-chart.html)
* [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart)
* [WilliamChart](https://github.com/diogobernardino/WilliamChart)
* [HelloCharts](https://github.com/lecho/hellocharts-android)

Paid:


* [aiCharts](http://www.artfulbits.com/products/android/aiCharts.aspx)
* [RChart](http://www.java4less.com/charts/chart.php?info=android) (pre Honeycomb - Api 11 UI)
* [ShinobiControls](http://www.shinobicontrols.com/android/shinobicharts/product-tour/) **
* [Steema TeeChart](https://play.google.com/store/apps/details?id=com.steema.teechart.android&hl=en) **
* [Orson Charts](http://www.object-refinery.com/orsoncharts/android/index.html) (3D charts for Android)
* [Telerik Rad Chart](http://www.telerik.com/android-ui/chart)

** - means I didn't try those so I can't really recommend it but other users suggested it..

Hope you and others will find it useful.. Cheers ;)
